What makes this tick

I've created this website mainly using the MEAN stack. Here is some information about that stack.

Mongo

Express

Angular

Node

Mongo

MongoDB is a document database with the scalability and flexibility that still provides you want with powerful querying and indexing.

I use it to store and retrieve my artwork.

More Information:

https://www.mongodb.com/what-is-mongodb

https://www.youtube.com/watch?v=CvIr-2lMLsk

Express

Express is a light-weight web application framework to help organize your web application into an MVC architecture on the server side.

I use it to communicate with my Mongo backend.

More Information:

https://expressjs.com/

https://www.youtube.com/watch?v=45dAt9Gz8rE

AngularJS

AngularJS is a framework for working with HTML in javascript. It helps add dynamic and responsive content through data-binding, and it works well with other libraries that I use, such as Bootstrap.

I use it to create the dynamic aspects of this site, such as the tab view on the left of this component.

More Information:

https://angularjs.org/

https://www.youtube.com/watch?v=zKkUN-mJtPQ

NodeJS

Node.js is an open-source, cross-platform JavaScript run-time environment that executes JavaScript code outside of a browser.

I use it to construct the application, and interact with Express and Angular.

More Information:

https://nodejs.org/en/about/

https://www.youtube.com/watch?v=uVwtVBpw7RQ
